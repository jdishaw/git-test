# Git Commands

## Basics 
### Make Changes
```bash
git add .
git commit -m "the commit"
```

### Get Updates
```bash
git pull
```

### Send Updates
```
git push
```

## Branching
### New Branch
```bash
git checkout -b new-branch
```

### Switch Branch
```bash
git checkout master
```

### List Branches on Local Computer
```
git branch
```

### List Branches on Local Computer and on Github
```
git branch -a 
```
